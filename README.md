## Architecture du Serveur:
`| - dist
| - src
| | - index.ts
| | - routes
|   | - hello.router.ts 
| - package.json
| - yarn.lock
| - tsconfig.json
| - .gitignore|`

Pour run on tape:  
`   cd Serveur/src/ 
    yarn install
    yarn start `
